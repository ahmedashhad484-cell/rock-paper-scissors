# Rock Paper Scissors

A fully playable Rock Paper Scissors game built with vanilla HTML, CSS, and JavaScript — no frameworks or libraries.

## Features

- Classic Rock Paper Scissors gameplay against the computer
- Live score tracking for both player and computer
- Round-by-round result messages (win, loss, or tie)
- First-to-five match logic with a win/loss screen
- Collapsible rules panel with a smooth fade animation using the Web Animations API
- "Play Again" button to reset scores and start a new match

## Tech used

- HTML5
- CSS3
- JavaScript (DOM manipulation, event listeners, Web Animations API)

## How to run

1. Clone or download this repository
2. Open `RPS.html` in any web browser
3. Click "Rules" to view how the game works, then choose Rock, Paper, or Scissors to play

No build steps or dependencies required.

## What I practiced

This project was built to practice:
- Managing game state (scores, results) in vanilla JavaScript
- Updating the DOM dynamically based on user interaction
- Conditional logic for game rules and win detection
- Basic UI animation without external libraries
